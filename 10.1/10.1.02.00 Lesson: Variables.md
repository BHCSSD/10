## 1. Creating Your Variables
Before you can move anything, you need a way to remember where an object is. you use **variables** to store this information.

Think of a variable as a labeled box. 
you declare (aka you make them) variables at the very top of the code so every function can see them.

```javascript
let xPos = 50
let yPos = 200
let speed = 2 
```

---

## 2. Making Shapes Move (`+=` and `-=`)
The `draw()` function runs about 60 times every second. 

To make anyhting move, you tell the computer: *"Every time you draw a frame, take the current position and add a little bit to it."*

* **`xPos += speed`**: adding the speed value to the position (Moving **Right**).
* **`xPos -= speed`**: subtracting the speed value (Moving **Left**).



```javascript
function draw() {
  background(220);
  
  circle(xPos, yPos, 50) //notice the variables instead of numbers for x,y
  
  xPos += speed // Update the position
}
```

---

## 3. The "Reset" Logic (The `if` Statement)
If you run the code above, your circle will eventually fly right off the screen and never come back.

you need an **if-statement** to check if the circle has gone too far.

> **The Logic:** "If the x value is bigger than the width of the canvas, set the x-position back to zero."

```javascript
if (xPos > 400) { 
  xPos = 0; //reset to the left side
}
```

---

## Final code
```javascript
let x = 0
let speed = 3

function setup() {
  createCanvas(400, 400);
}//end of setup

function draw() {
  background(50);
  
  fill(0, 255, 150)
  ellipse(x, 200, 50, 50)
  
  //1. Move the shape
  x += speed
  
  //2. Check if it left the screen
  if (x > 400) {
    x = -25 //Start off screen for a better transition
  }//end of if

}//end of draw
```
